# 隱私權政策

**最後更新日期：2026 年 3 月 13 日**

本隱私權政策說明 GForm Bouncer（以下簡稱「本服務」）如何蒐集、使用及保護您的個人資料。

## 1. 蒐集的資料

本服務僅蒐集及處理以下資料：

| 資料類型 | 來源 | 用途 |
|---|---|---|
| Discord 使用者 ID | 使用者在 Google 表單中填寫 | 用於比對 Discord 伺服器成員並指派角色 |
| Discord 使用者名稱 | Discord API（僅用於日誌紀錄） | 於操作日誌中顯示處理結果 |
| Google 表單回覆 ID | Google Forms API | 避免重複處理同一筆回覆 |

**本服務不會蒐集：**
- 電子郵件地址
- 密碼或其他認證資訊
- 訊息內容
- 語音通話資料
- IP 位址
- 裝置資訊

## 2. 資料使用方式

您的資料僅用於以下目的：

- **角色指派**：將您在表單中填寫的 Discord ID 與 Discord 伺服器成員進行比對，以自動指派角色。
- **重複檢查**：記錄已處理的回覆 ID，避免重複指派。
- **操作日誌**：在伺服器端日誌中記錄指派結果，供管理員除錯使用。

## 3. 資料儲存

- 已處理的回覆 ID 僅儲存於應用程式的**執行時期記憶體**中，Bot 重啟後即清除。
- 本服務**不使用資料庫**，不會將您的個人資料持久化儲存。
- 操作日誌可能保留在伺服器主機的標準輸出中，具體保留期間取決於主機的日誌管理設定。

## 4. 資料分享

本服務**不會**將您的資料分享、出售或提供給任何第三方。

本服務與以下第三方平台互動，但僅限於其 API 的標準使用範圍：

- **Discord API**：讀取伺服器成員清單、指派角色。
- **Google Forms API**：讀取表單回覆內容。

請同時參閱這些平台各自的隱私權政策：
- [Discord 隱私權政策](https://discord.com/privacy)
- [Google 隱私權政策](https://policies.google.com/privacy)

## 5. 資料安全

我們採取合理的安全措施來保護您的資料：

- API 金鑰及認證資訊儲存於環境變數中，不會寫入原始碼。
- 本服務僅請求運作所需的最低權限（讀取表單回覆、管理角色）。
- 所有 API 通訊皆透過 HTTPS 加密傳輸。

## 6. 使用者權利

您擁有以下權利：

- **存取權**：您可以向伺服器管理員查詢本服務是否已處理您的資料。
- **刪除權**：由於本服務不持久化儲存個人資料，Bot 重啟後執行時期資料即自動清除。如需移除已指派的角色，請聯繫伺服器管理員。
- **退出權**：您可以選擇不填寫 Google 表單，即不會被本服務處理。

## 7. 兒童隱私

本服務不會刻意蒐集 13 歲以下兒童的個人資料。若您未滿 13 歲（或您所在地區使用 Discord 的最低年齡要求），請勿使用本服務。

## 8. 政策修改

我們保留隨時修改本隱私權政策的權利。修改後的政策將於本頁面公佈後生效。建議您定期查閱本政策以瞭解最新內容。

## 9. 聯絡方式

如對本隱私權政策有任何疑問或要求，請透過 Discord 伺服器聯繫管理員。
