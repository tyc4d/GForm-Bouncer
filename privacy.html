隱私權政策

最後更新日期：2026 年 3 月 13 日

本隱私權政策說明 GForm Bouncer 如何蒐集、使用及保護您的個人資料。

1. 蒐集的資料

資料類型 來源 用途
Discord 使用者 ID Google 表單 指派角色
Discord 使用者名稱 Discord API 日誌紀錄
Google 表單回覆 ID Google Forms API 避免重複處理
我們不會蒐集： 電子郵件、密碼、訊息內容、語音通話資料、IP 位址或裝置資訊。

2. 資料儲存

  • 已處理的回覆 ID 僅儲存於執行時期記憶體中，Bot 重啟後即清除。
  • 本服務不使用資料庫，不會將您的個人資料持久化儲存。

3. 資料安全

我們採取合理的安全措施：API 金鑰儲存於環境變數中，僅請求最低權限，且所有通訊皆透過 HTTPS 加密傳輸。

4. 使用者權利

您擁有存取權與刪除權。由於本服務不持久化儲存資料，Bot 重啟後執行時期資料即自動清除。

如對本隱私權政策有任何疑問，請透過 Discord 伺服器聯繫管理員。
